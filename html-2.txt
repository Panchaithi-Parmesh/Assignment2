Below mentioned style attributes to apply CSS in any HTML page
A: HTML Background Color
<!DOCTYPE html>
<html>
<head>
<style>
body {
  background-color: coral;
}
</style>
</head>
<body>
<h1>This is first web site and <b> we are learning </b></h1>
</body>
</html>



B: HTML Text Color
<!DOCTYPE html>
<html>
<head>
<style>
body {
  background-color: coral;
}
</style>
</head>
<body>
<h1 style="color:black;">This is a heading</h1>
</body>
</html>
